## 如何開始

1.  透過bower安裝套件
    ```
    bower install
    ```
2.  透過npm安裝套件
    ```
    npm install
    ```
3.  預覽開發程式碼
    ```
    gulp serve
    ```
4.  產出靜態檔
    ```
    gulp
    ```
5.  預覽靜態檔
    ```
    gulp serve:dist
    ```
